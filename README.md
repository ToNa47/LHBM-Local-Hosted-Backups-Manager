# LHBM (Local Hosted Backups Management)

Is a program i imagined and work to for those who's lazy to backup daily or something
LHBM need futher updates that tracks automatically if a file was changed/deleted

How to track:
-Add your chosen file to track to the choice
-Add your backups destination Folder 
-Set your desired duration for the tracking
-Done

How to recall:
-Locate your backups Folder
-Find the zips folder
-Choose which time you want to recall to
-Get your zip away
-Unzip
-Done
